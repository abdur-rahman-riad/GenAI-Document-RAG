## Project Description
**VOID is an advanced Retrieval-Augmented Generation (RAG) chatbot built with Streamlit, LangChain, ChromaDB, HuggingFace Embeddings, and Google Gemini 2.5 Flash. It allows users to upload PDF documents, automatically extract and process the content using OCR technology, convert it into searchable vector embeddings, and then ask natural language questions about the document. The system provides accurate, context-aware answers based strictly on the uploaded document content.**

## Key Features

- **Smart PDF Processing**  
  Automatic text extraction with OCR fallback for scanned or complex documents.
  
- **Advanced RAG Architecture**  
  Efficient vector embeddings with semantic similarity search for accurate answers.

- **Beautiful Chat Interface**  
  Modern, messenger-style UI with an elegant purple theme.

- **Powered by Gemini 2.5**  
  Cost-efficient large language model delivering accurate and contextual responses.

- **Session-Based Storage**  
  Secure, temporary document storage isolated per user session.

- **Real-Time Processing**  
  Instant document analysis, chunking, and embedding for fast interaction.

---

## Tech Stack

- **Frontend:** Streamlit  
- **RAG Framework:** LangChain  
- **LLM:** Google Gemini 2.5 Flash  
- **Vector Database:** ChromaDB  
- **Embeddings:** HuggingFace (`sentence-transformers/all-MiniLM-L6-v2`)
- **PDF Processing:** PyPDF, pdf2image, Tesseract OCR  
- **Programming Language:** Python 3.9+
